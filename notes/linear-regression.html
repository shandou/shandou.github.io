Task description

Simulate data of 5 features x 10000 rows with Gaussian noise standard deviation of 10. Write out the training and testing procedure.

import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt
from sklearn.model_selection import train_test_split
from sklearn.metrics import r2_score
from sklearn.linear_model import LinearRegression
from sklearn.preprocessing import MinMaxScaler


# ======= Part 1: Data simulation =======
# Specify feature dimensions
n_samples, n_features = 100000, 5
sigma = 10
X = np.random.rand(n_samples, n_features)

# Specify synthetic model parameters in weights and bias (intercept)
w, bias = np.array([20, 20, 30, 50, 100]), 15
# Gnerate Gaussian residual as required by the task
error_residual = np.random.normal(0, sigma, n_samples)
# Perform dot product to generate prediction target
y = np.dot(w, X.T) + bias + error_residual


# ======= Part 2: Modeling =======
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3)
scaler = MinMaxScaler()
X_train_scaled = scaler.fit_transform(X_train)
X_test_scaled = scaler.transform(X_test)
model = LinearRegression()
model.fit(X_train_scaled, y_train)
y_pred = model.predict(X_test)
print(r2_score(y_test, y_pred))
# Out: 0.9224441433425903


# Manually implement r_squared
explained_variance = np.sum((y_pred - y_test.mean())**2)
residual_variance = np.sum((y_pred - y_test)**2)
total_variance = explained_variance + residual_variance
r2_calculated = explained_variance / total_variance
# Out: 0.922594062184237

Linear regression assumptions

  1. Error-free predictors: This is an important assumption and also where I made the grave mistake (Image source: GeostatsGuy Lectures)

  2. Constant variance: The error is homoscedestic

  3. Relationship between independent and dependent variables is liinear
  4. Errors are independent of each other (otherwise, this would be an indication that there are additional relationships now accounted for by the model)
  5. No multicolinearity: Features are not correlated with each other (with redundany)
