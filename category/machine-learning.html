Simply Explained: What is Machine Learning?

Published: Sat 20 March 2021
By Shan Dou

In Machine Learning.

tags: machine-learning

Despite the hype and frequent misunderstanding of the phrase "machine learning", machine learning is about building statistical models for the data. For example, a support vector machine binary classifier draws a straight line between data of positive and negative classes:

  • The model is a quantitative statement that says: With this model, a straight line is drawn to separate the two classes. In other words, the model could be viewed as a placeholder that is described by the features used, how features will be combined by using a given algorithm ( hyperparameters describe the high-level structure). In some sense, the model can be compared to a computer with its basic operating system and applications installed but not yet customized to individual use cases
  • Model parameters are the tunable parameters that allow the model to adapt to the data we are trying to model. For the SVM example, they are the parameters that describe the specific location and orientation of the straightline decision boundary
  • The learning part of machine learning entails learning about the optimal values of these model parameters from the data

What makes machine learning powerful has less to do with its often hyped "machine intelligence". But rather, it allows fuzzy, automatic generalization (as opposed to hard-coded rules) to much larger datasets in high dimensions.
